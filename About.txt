1. Компонет является функцией, которая возвращает jsx разметк
2. Все компоненты выносятся в отдельные файлы. Файлы помещаются в отдельные папки.
3. В каждой папке кроме файла jsx находится файл формата module.css или module.scss
4. module.scss нужен для того, чтобы создать уникальные css правила, которые будут действовать только
    на конкретный jsx файл, к которму этот модуль подключен
5. Готовый компонент эскортируется export default  и import куда надо. В данном случае напрямую в App.
6. App является основным файлом-сборщиком всех компонентов. Уже далее App рендерится в index.js    
7. Весь синтаксис указан в файлах
